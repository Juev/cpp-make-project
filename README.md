# Make CPP project with CMake

Program for creating simple CPP project with CMake.

For building:

    $ mkdir build
    $ cd build
    $ cmake ..
    $ make

For using:

    $ make-project projectName

if projectName skipped, used template name.

Program will create simple structure:

    |- CMakeList.txt
    |- source
    |- |- main.cpp
